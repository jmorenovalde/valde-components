# ValdeComponents

This is a sample project on [Angular 15](https://angular.io) to learn how to use [Storybook](https://storybook.js.org/docs/angular/get-started/introduction).

I added some components with several functions to show on the stories and a complex component (login) to check the test and the actions on the Storybook.

## Install

For the Angular project, you can run `npm install`.

## Run

This _Angular project_ has some projects to show. The application shows the use of components as `Angular Project`.

```bash
npm run start
```

2. The _Storybook application_ shows the components with the _Storybook_ interface

```bash
npm run storybook
```

3. The _Compodoc application_ shows the documentation of the _Angular Application_ with the _Compodoc interface_.

```bash
npm run compodoc
```

4. The _Jest test suit_.

```bash
npm run test
```
