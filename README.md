# ValdeComponents

This is a sample project on [Angular 14](https://angular.io) to learn how to use [Storybook](https://storybook.js.org/docs/angular/get-started/introduction).

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 14.0.4.

## Install

As every _Angular project_ you can run `npm install`.

## Run

This _Angular project_ has some projects to show.

1. The application to show the use of components as `Angular Project`.

```bash
npm run start
```

2. The _Storybook application_ to show the components with the _Storybook_ interface

```bash
npm run storybook
```

3. The _Compodoc application_ to show the documentation of the _Angular Application_ with the _Compodoc interface_.
```bash
npm run compodoc
```
